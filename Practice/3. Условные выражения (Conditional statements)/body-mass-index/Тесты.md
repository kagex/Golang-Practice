# Запуск тестов

Для проверки корректности работы программы используются автоматические тесты, написанные с использованием стандартного пакета `testing`.

## 1. Базовый запуск

Чтобы запустить все тесты и увидеть подробный отчет о каждом проверенном сценарии, выполните в терминале команду:

```bash
go test -v
```

* **`go test`** — стандартная команда Go для поиска и запуска файлов с суффиксом `_test.go`.
* **`-v` (verbose)** — флаг подробного вывода. Он заставляет тестировщик печатать в консоль статус каждого отдельного теста и подтеста (из `t.Run`), а не просто итоговое `PASS` или `FAIL`.

## 2. Дополнительные полезные команды

**Запуск только тестов BMI**
```bash
go test -v -run TestBMI
```

**Запуск только тестов ошибок**
```bash
go test -v -run TestBMIErrors
```

## 3. Ожидаемый результат

### ✅ Если все тесты пройдены успешно:
В консоли вы увидите подробный список всех сценариев (недостаточный вес, нормальный вес, избыточный вес, ожирение, ошибки ввода) и итоговое сообщение `PASS`.

```text
=== RUN   TestBMI
=== RUN   TestBMI/недостаточный_вес
=== RUN   TestBMI/нормальный_вес_среднее
=== RUN   TestBMI/избыточный_вес
=== RUN   TestBMI/ожирение
=== RUN   TestBMIErrors
=== RUN   TestBMIErrors/отрицательный_вес
...
--- PASS: TestBMI (2.56s)
    --- PASS: TestBMI/недостаточный_вес (0.65s)
    --- PASS: TestBMI/нормальный_вес_среднее (0.66s)
    --- PASS: TestBMIErrors/отрицательный_вес (0.67s)
...
PASS
ok      body-mass-index       8.333s
```

### ❌ Если в коде есть ошибка:
Тест, который не прошел, будет помечен как `FAIL`, а вы увидите сообщение об ошибке, которое вы сгенерировали через `t.Errorf`.

```text
=== RUN   TestBMI/нормальный_вес_среднее
    main_test.go:52: Expected "Ваш ИМТ: 22.86\nКатегория: Нормальный вес", got "Ваш ИМТ: 25.00\nКатегория: Избыточный вес"
--- FAIL: TestBMI (2.10s)
    --- FAIL: TestBMI/нормальный_вес_среднее (0.14s)
FAIL
exit status 1
FAIL    body-mass-index       2.150s
```

## ⚠️ Требования для запуска

1. Файлы `main.go` и `main_test.go` должны находиться в **одной директории**.
2. Оба файла должны принадлежать пакету `main` (директива `package main` в самом начале файлов).
3. Запускать команду `go test -v` нужно из терминала, находясь в папке с этими файлами.
