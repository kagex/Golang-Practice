# Поиск по названию товара

## Условие задачи

Напишите программу, которая будет выполнять поиск товара по его названию. Пользователь должен ввести название товара или его часть, после чего программа должна вывести в консоль цену данного товара в формате:

```
Название товара: цена товара
```

## Доступные товары и их цены

| Товар            | Цена  |
|------------------|-------|
| Клавиатура JZ9   | 19200 |
| Наушники N45     | 9600  |
| Смартфон S10     | 55000 |

## Требования к поиску

1. **Частичное совпадение** — поиск должен работать по подстроке. Например, ввод `сМАРТ` должен найти `Смартфон S10`.
2. **Нечувствительность к регистру** — `наУШНИКИ n45`, `наушники n45` и `НАУШНИКИ N45` должны находить один и тот же товар.

> 💡 **Важное уточнение:** решить задачу можно намного проще, если использовать Map для хранения пар "название товара - цена товара". Но по условию задачи мы должны использовать самые базовые операторы и методы.
## Выходные данные

- Если товар найден, вывести:
  ```
  Название товара: цена товара
  ```

- Если товара нет в списке, вывести:
  ```
  Товар "название_введенного_товара" не найден.
  ```

## Примеры работы программы

### Пример 1: Точный поиск (с другим регистром)
```text
Введите название товара: наУШНИКИ n45
Наушники N45: 9600
```

### Пример 2: Частичный поиск
```text
Введите название товара: сМАРТ
Смартфон S10: 55000
```

### Пример 3: Товар не найден
```text
Введите название товара: планшет
Товар "планшет" не найден.
```

> 💡 **Важное уточнение:** программа вернет только первое вхождение. Всё в рамках условия задачи.
