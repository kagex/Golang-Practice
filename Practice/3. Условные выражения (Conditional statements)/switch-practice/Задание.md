## Разбираем мусор

У нас есть партнеры, которые вечно присылают что-то случайное. Необходимо "отсортировать" все данные.

В переменной `val` типа `any`, лежит какое-то значение, необходимо использовать type switch, чтобы определить, является ли значение внутри переменной `int`, `string`, `float64` или `bool`.

Выведите соответствующее сообщение для каждого типа, пример строки для вывода:
"В переменной val находится тип [ТИП]."
В случае, если в переменной будет что-то иное, необходимо вывести:
"В переменной val находится неизвестный тип данных."

### Пример работы

```go
var val any = 42
// Вывод: В переменной val находится тип int.

var val any = "привет"
// Вывод: В переменной val находится тип string.

var val any = 3.14
// Вывод: В переменной val находится тип float64.

var val any = true
// Вывод: В переменной val находится тип bool.

var val any = []int{1, 2, 3}
// Вывод: В переменной val находится неизвестный тип данных.
```
