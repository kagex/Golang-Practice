# Задание: Сумма цифр числа

Это практическое задание на рекурсию

Вам необходимо написать рекурсивную функцию `sumOfDigits`, которая принимает одно целое число и возвращает сумму его цифр. Функция должна работать как с положительными, так и с отрицательными числами.

## Пример работы функции

```go
sumOfDigits(123)   // Возвращает 6 (1 + 2 + 3)
sumOfDigits(-456)  // Возвращает 15 (4 + 5 + 6)
sumOfDigits(0)     // Возвращает 0
```
