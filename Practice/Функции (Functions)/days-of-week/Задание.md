# Создание типа для дней недели

Нужно поработать с днями недели. Так как дни — это определенный набор значений, нам нужно создать новый тип `Day`, который будет представлять дни недели (от 1 до 7 включительно). Необходимо использовать `iota` для автоматического присвоения значений дням недели. 

Названия дней недели должны быть следующими:

- `Monday`
- `Tuesday`
- `Wednesday`
- `Thursday`
- `Friday`
- `Saturday`
- `Sunday`

Реализуйте функцию `isWeekend`, которая принимает `Day` и возвращает `true`, если это выходной (суббота или воскресенье), и `false` в противном случае.

### Тестовые данные

- `isWeekend(6)` → получим значение `true`
- `isWeekend(3)` → получим значение `false`
