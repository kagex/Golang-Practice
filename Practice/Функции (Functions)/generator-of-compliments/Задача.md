# Генератор комплиментов

Каждому разработчику время от времени нужна поддержка. Давайте напишем функцию, которая будет поднимать настроение пользователям нашего приложения.

Вам необходимо реализовать функцию `generateCompliment(name string) string`, которая принимает имя человека и возвращает случайный комплимент из заранее предопределенного списка, подставляя переданное имя.

Функция должна случайно выбирать один из трех следующих вариантов комплиментов:

- "Ты великолепен, [имя]!"
- "У тебя потрясающая улыбка, [имя]!"
- "Ты вдохновляешь, [имя]!"
