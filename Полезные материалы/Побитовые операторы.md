# Побитовые операторы в Go

## Обзор

Побитовые операторы в Go позволяют выполнять манипуляции с отдельными битами целочисленных значений. Они работают напрямую с двоичным представлением чисел и часто используются для оптимизации, работы с флагами (bitset), сетевыми протоколами и низкоуровневым программированием.

## Доступные побитовые операторы

### Побитовое И (`&`)

Выполняет логическое И для каждой пары битов. Результат = 1 только если оба бита равны 1.

```go
a := 60  // 0011 1100
b := 13  // 0000 1101
c := a & b  // 0000 1100 = 12
```

### Побитовое ИЛИ (`|`)

Выполняет логическое ИЛИ для каждой пары битов. Результат = 1 если хотя бы один бит равен 1.

```go
a := 60  // 0011 1100
b := 13  // 0000 1101
c := a | b  // 0011 1101 = 61
```

### Побитовое исключающее ИЛИ (XOR, `^`)

Результат = 1 если биты различаются (один 0, другой 1).

```go
a := 60  // 0011 1100
b := 13  // 0000 1101
c := a ^ b  // 0011 0001 = 49
```

### Побитовое НЕ (`^` или `&^`)

В Go оператор `^` используется как **побитовое НЕ** для одного операнда (инверсия всех битов):

```go
a := 60      // 0011 1100
c := ^a      // 1100 0011 (инверсия)
```

Для двух операндов `^` — это XOR (см. выше).

Для **очистки битов** (AND NOT) используется `&^`:
- `a &^ b` очищает те биты `a`, которые установлены в `b`

```go
a := 60  // 0011 1100
b := 13  // 0000 1101
c := a &^ b  // 0011 0000 = 48
```

### Сдвиг влево (`<<`)

Сдвигает биты влево на указанное количество позиций. Эквивалентно умножению на 2^n.

```go
a := 5       // 0000 0101 = 5
b := a << 2  // 0001 0100 = 20 (5 * 2^2)
```

### Сдвиг вправо (`>>`)

Сдвигает биты вправо на указанное количество позиций. Эквивалентно целочисленному делению на 2^n.

```go
a := 20      // 0001 0100 = 20
b := a >> 2  // 0000 0101 = 5 (20 / 2^2)
```

Для знаковых целых чисел (`int`) используется арифметический сдвиг (сохраняется знак). Для беззнаковых (`uint`) — логический сдвиг (заполняется нулями).

## Основные применения

### 1. Работа с флагами (Bitmask)

Полный пример работы с правами доступа:

```go
package main

import (
	"fmt"
)

const (
	Read    byte = 1 << 0 // 00000001 (1)
	Write   byte = 1 << 1 // 00000010 (2)
	Execute byte = 1 << 2 // 00000100 (4)
	Delete  byte = 1 << 3 // 00001000 (8)
)

func main() {
	var userPerms byte = Read | Write
	fmt.Printf("1. Изначальные права: %08b\n", userPerms)

	hasRead := (userPerms & Read) != 0
	hasExecute := (userPerms & Execute) != 0
	fmt.Printf("2. Есть право на чтение? %t\n", hasRead)
	fmt.Printf("   Есть право на выполнение? %t\n", hasExecute)

	userPerms |= Execute
	fmt.Printf("3. Добавлено выполнение:  %08b\n", userPerms)

	userPerms &^= Write
	fmt.Printf("4. Отозвано право записи: %08b\n", userPerms)

	userPerms ^= Delete
	fmt.Printf("5. Переключено удаление (добавлено): %08b\n", userPerms)
	
	userPerms ^= Delete
	fmt.Printf("   Переключено удаление (убрано):    %08b\n", userPerms)
}
```

Выход:
```
1. Изначальные права: 00000011
2. Есть право на чтение? true
   Есть право на выполнение? false
3. Добавлено выполнение:  00000111
4. Отозвано право записи: 00000101
5. Переключено удаление (добавлено): 00001101
   Переключено удаление (убрано):    00000101
```

**Ключевые операции:**
- **Установка**: `perms |= Read` (побитовое ИЛИ)
- **Проверка**: `(perms & Read) != 0` (побитовое И)
- **Очистка**: `perms &^= Read` (AND NOT)
- **Переключение**: `perms ^= Read` (XOR)

### 2. Проверка четности/нечетности

```go
isEven := (n & 1) == 0   // четное, если младший бит = 0
isOdd := (n & 1) == 1    // нечетное, если младший бит = 1
```

### 3. Быстрое умножение/деление на степени двойки

```go
x * 8    == x << 3
x / 16   == x >> 4
```

### 4. Упаковка/распаковка данных

```go
// Упаковка двух 8-битных значений в 16-битное
byte1 := byte(0x12)
byte2 := byte(0x34)
value := uint16(byte1)<<8 | uint16(byte2)  // 0x1234

// Распаковка
byte1_unpacked := byte(value >> 8)   // 0x12
byte2_unpacked := byte(value & 0xFF) // 0x34
```

## Таблица операторов

| Оператор | Название | Пример | Описание |
|----------|----------|--------|----------|
| `&` | Побитовое И | `a & b` | Логическое И: оба бита = 1 |
| `\|` | Побитовое ИЛИ | `a \| b` | Логическое ИЛИ |
| `^` | XOR / Побитовое НЕ | `a ^ b` / `^a` | Исключающее ИЛИ / Инверсия |
| `&^` | AND NOT | `a &^ b` | Очистка битов |
| `<<` | Сдвиг влево | `a << n` | Умножение на 2^n |
| `>>` | Сдвиг вправо | `a >> n` | Деление на 2^n |

## Важные замечания

1. Побитовые операторы работают только с целочисленными типами (`int`, `uint`, `int8`, `uint8`, `int16`, `uint16`, `int32`, `uint32`, `int64`, `uint64`, `rune`, `byte`, `uintptr`)
2. Строки и числа с плавающей точкой не поддерживают побитовые операции
3. Сдвиг вправо для `int` (знаковый) использует арифметический сдвиг (сохраняет знак)
4. Сдвиг вправо для `uint` (беззнаковый) использует логический сдвиг (заполняет нулями)
5. Сдвиг на 0 не меняет значение: `n << 0 == n >> 0 == n`

## Примеры задач

### Подсчет количества единичных битов

```go
func countBits(n uint) int {
	count := 0
	for n != 0 {
		count += int(n & 1)
		n >>= 1
	}
	return count
}
```

### Проверка, является ли число степенью двойки

```go
func isPowerOfTwo(n int) bool {
	return n > 0 && (n&(n-1)) == 0
}
```

### Получение/установка бита по индексу

```go
// Получить бит на позиции i
bit := (n >> i) & 1

// Установить бит на позиции i
n |= (1 << i)

// Сбросить бит на позиции i
n &^= (1 << i)
```
