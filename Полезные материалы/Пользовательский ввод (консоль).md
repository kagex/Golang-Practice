# Пользовательский ввод в Go (консоль)

Читать, что вводит пользователь в консоли, можно разными методами.

## Методы ввода

- `fmt.Scanln()`, `fmt.Scan()`, `fmt.Scanf()` — считывание слов в отдельные переменные
- `bufio.Reader`, `bufio.Scanner` — считывание строки текста или одного символа

Выбор метода зависит от задачи и личных предпочтений.


## fmt.Scanln()

Чтение строки. Важно! Вводить нужно ровно столько значений, сколько переменных указано. При недостатке ввода будет ошибка.

### `fmt.Scanln(a ...any) (n int, err error)`

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    fmt.Println("Введите текст:")
    var w1, w2, w3 string
    n, err := fmt.Scanln(&w1, &w2, &w3)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Количество считанных элементов: %d\n", n)
    fmt.Printf("Считанные элементы: %s %s %s\n", w1, w2, w3)
}
```

**Вывод:**

```
Введите текст:
Пр ив ет
Количество считанных элементов: 3
Считанные элементы: Пр ив ет
```

## fmt.Scan()

Чтение нескольких значений. Игнорирует переносы строк (считывает данные, пока не заполнит все переданные переменные). Требует ровно столько аргументов, сколько указано переменных.

### `fmt.Scan(a ...any) (n int, err error)`

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    fmt.Println("Введите текст:")
    var w1, w2, w3 string
    n, err := fmt.Scan(&w1, &w2, &w3)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Количество считанных элементов: %d\n", n)
    fmt.Printf("Считанные элементы: %s %s %s\n", w1, w2, w3)
}
```

**Вывод:**

```
Введите текст:
Пр
ив
ет
Количество считанных элементов: 3
Считанные элементы: Пр ив ет
```

## bufio.Reader

Для чтения одной строки текста.

> **Важно!** Метод `ReadString('\n')` сохраняет символ `\n` в итоговой строке. Поэтому при выводе через `fmt.Printf("%s\n", line)` получается дополнительный перенос.

### `reader.ReadString(delim byte) (string, error)`

```go
package main

import (
    "bufio"
    "fmt"
    "log"
    "os"
)

func main() {
    fmt.Println("Введите текст:")
    reader := bufio.NewReader(os.Stdin)
    line, err := reader.ReadString('\n')
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Считанная строка: %s\n", line)
}
```

**Вывод:**

```
Введите текст:
Пр ив ет
Считанная строка: Пр ив ет

```

## bufio.Scanner

Альтернатива для чтения одной строки текста.

> **Важно!** Метод `scanner.Text()` автоматически удаляет терминальный символ `\n`. Работает аналогично `Console.ReadLine()` из C# — возвращает строку целиком, но без завершающего переноса.

### `scanner.Text() string`

```go
package main

import (
    "bufio"
    "fmt"
    "log"
    "os"
)

func main() {
    fmt.Println("Введите текст:")
    scanner := bufio.NewScanner(os.Stdin)
    scanner.Scan()
    if err := scanner.Err(); err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Считанная строка: %s\n", scanner.Text())
}
```

**Вывод:**

```
Введите текст:
Пр ив ет
Считанная строка: Пр ив ет
```

## Чтение нескольких строк

### С использованием bufio.Reader

```go
package main

import (
    "bufio"
    "fmt"
    "log"
    "os"
    "strings"
)

func main() {
    fmt.Println("Введите текст:")
    reader := bufio.NewReader(os.Stdin)
    var lines []string
    for {
        line, err := reader.ReadString('\n')
        if err != nil {
            log.Fatal(err)
        }
        if len(strings.TrimSpace(line)) == 0 {
            break
        }
        lines = append(lines, line)
    }
    fmt.Println("Вывод:")
    for _, l := range lines {
        fmt.Printf(l)
    }
}
```

**Вывод:**

```
Введите текст:
Пр ив ет
из golang

Вывод:
Пр ив ет
из golang
```

### С использованием bufio.Scanner

```go
package main

import (
    "bufio"
    "fmt"
    "log"
    "os"
)

func main() {
    fmt.Println("Введите текст:")
    scanner := bufio.NewScanner(os.Stdin)
    var lines []string
    for {
        scanner.Scan()
        line := scanner.Text()
        if len(line) == 0 {
            break
        }
        lines = append(lines, line)
    }
    if err := scanner.Err(); err != nil {
        log.Fatal(err)
    }
    fmt.Println("Вывод:")
    for _, l := range lines {
        fmt.Println(l)
    }
}
```

**Вывод:**

```
Введите текст:
Пр ив ет
из golang

Вывод:
Пр ив ет
из golang
```

## Чтение одного символа

### С использованием bufio.Reader

```go
package main

import (
    "bufio"
    "fmt"
    "log"
    "os"
)

func main() {
    fmt.Println("Введите текст:")
    reader := bufio.NewReader(os.Stdin)
    char, _, err := reader.ReadRune()
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Считанный символ: %c\n", char)
}
```

**Вывод:**

```
Введите текст:
Привет
Считанный символ: П
```

### С использованием fmt.Scanf()

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    fmt.Println("Введите символ:")
    var char rune
    _, err := fmt.Scanf("%c", &char)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("Считанный символ: %c\n", char)
}
```

**Вывод:**

```
Введите символ:
П
Считанный символ: П
```

## Чтение одного числа

### С использованием fmt.Scanf()

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    fmt.Println("Введите число:")
    var number int64
    _, err := fmt.Scanf("%d", &number)

    }
    fmt.Printf("Считанное число: %d\n", number)
}

**Вывод:**

```
Введите число:
24

## Сравнение методов

| Метод | Пример | Особенности | Когда использовать |
|-------|--------|-------------|-------------------|
| `fmt.Scanln()` | `fmt.Scanln(&a, &b)` | Считывает до переноса строки, строгое количество аргументов | Для простого ввода нескольких значений, когда нужно строгое количество |
| `fmt.Scan()` | `fmt.Scan(&a, &b)` | Игнорирует переносы строк (считывает данные, пока не заполнит все переданные переменные) | Для ввода переменного количества значений |
| `fmt.Scanf()` | `fmt.Scanf("%d", &n)` | Форматированный ввод по шаблону | Когда нужен специфический формат ввода (числа, символы) |
| `bufio.Reader` | `reader.ReadString('\n')` | Полный контроль, считывает до указанного символа | Когда нужен полный контроль над процессом ввода |
| `bufio.Scanner` | `scanner.Text()` | Простой интерфейс, удобен для построчного чтения | Для простого построчного ввода, самый удобный для большинства задач <sup>1</sup> |

> **Примечание 1.** По умолчанию `bufio.Scanner` имеет лимит на размер токена — 64 КБ (65536 байт). Если строка длиннее, сканер выдаст `bufio.ErrTooLong`. В отличие от него, `bufio.Reader` с `ReadString` может читать строки любой длины.
