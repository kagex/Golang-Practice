# Спецификаторы формата в Go

## Обзор

Спецификаторы формата используются в функциях форматирования вывода Go для контроля над тем, как значения отображаются при печати. Они применяются в функциях `fmt.Printf()`, `fmt.Sprintf()`, `fmt.Errorf()` и других.

## Основные спецификаторы

### Целые числа

| Спецификатор | Описание | Пример (42) |
|-------------|----------|-------------|
| `%d` | Десятичное целое число (основание 10) | `42` |
| `%x` | Шестнадцатеричное число (строчные буквы) | `2a` |
| `%X` | Шестнадцатеричное число (заглавные буквы) | `2A` |
| `%o` | Восьмеричное число (основание 8) | `52` |
| `%b` | Двоичное число (основание 2) | `101010` |
| `%c` | Символ (rune) | `*` |
| `%U` | Unicode-код точки | `U+002A` |

### Числа с плавающей точкой

| Спецификатор | Описание | Пример (3.14159) |
|-------------|----------|-----------------|
| `%f` | Десятичная запись | `3.141590` |
| `%e` | Экспоненциальная запись (строчные) | `3.141590e+00` |
| `%E` | Экспоненциальная запись (заглавные) | `3.141590E+00` |
| `%g` | `%e` или `%f` (выбирает короче) | `3.14159` |
| `%G` | `%E` или `%f` (выбирает короче) | `3.14159` |

### Строки и байты

| Спецификатор | Описание | Пример |
|-------------|----------|--------|
| `%s` | Строка | `Hello` |
| `%q` | Строка в кавычках | `"Hello"` |
| `%x` | Строка в шестнадцатеричном формате | `48656c6c6f` |
| `%X` | То же, что `%x`, но заглавные буквы | `48656C6C6F` |
| `%v` | Значение в формате по умолчанию | `Hello` |

### Указатели и адреса

| Спецификатор | Описание | Пример |
|-------------|----------|--------|
| `%p` | Адрес указателя в шестнадцатеричном виде | `0x1040a120` |
| `%T` | Тип значения | `int` |

### Общие спецификаторы

| Спецификатор | Описание | Пример |
|-------------|----------|--------|
| `%v` | Значение в формате по умолчанию | `42` |
| `%+v` | Значение с именами полей для структур | `Name:Ivan Age:25` |
| `%#v` | Значение в Go-синтаксисе | `main.Person{Name:"Ivan", Age:25}` |
| `%#p` | Адрес указателя в Go-синтаксисе | `&0x1040a120` |

## Флаги форматирования

### Ширина поля

Ширина поля определяет минимальное количество символов для вывода.

```go
fmt.Printf("'%d'\n", 42)        // '42'
fmt.Printf("'%5d'\n", 42)       // '   42' (3 пробела перед числом)
fmt.Printf("'%05d'\n", 42)      // '00042' (ноль заполняет пробелы)
fmt.Printf("'%-5d'\n", 42)      // '42   ' (левостороннее выравнивание)
```

### Точность

Для чисел с плавающей точкой — количество знаков после запятой.

```go
fmt.Printf("%.2f\n", 3.14159)   // 3.14
fmt.Printf("%.0f\n", 3.14159)   // 3
fmt.Printf("%.10f\n", 3.14)     // 3.1400000000
```

Для строк — максимальное количество символов для вывода.

```go
fmt.Printf("%.5s\n", "Hello, World!") // Hello
```

### Дополнительные флаги

| Флаг | Описание | Пример |
|------|----------|--------|
| `+` | Всегда показывать знак для чисел | `+42` |
| `-` | Левостороннее выравнивание | `42   ` |
| `#` | Альтернативная форма (префиксы, кавычки) | `0x2a`, `"text"` |
| ` ` (пробел) | Пробел перед положительными числами | ` 42` |
| `0` | Заполнение нулями (вместо пробелов) | `00042` |

## Примеры комбинаций

### Целые числа

```go
n := 42

fmt.Printf("%d\n", n)          // 42
fmt.Printf("%+d\n", n)         // +42
fmt.Printf("%5d\n", n)         // '   42'
fmt.Printf("%05d\n", n)        // '00042'
fmt.Printf("%x\n", n)          // 2a
fmt.Printf("%X\n", n)          // 2A
fmt.Printf("%#x\n", n)         // 0x2a
fmt.Printf("%08b\n", n)        // 00101010
```

### Числа с плавающей точкой

```go
f := 3.14159

fmt.Printf("%f\n", f)          // 3.141590
fmt.Printf("%.2f\n", f)        // 3.14
fmt.Printf("%e\n", f)          // 3.141590e+00
fmt.Printf("%g\n", f)          // 3.14159
fmt.Printf("%5.2f\n", f)       // ' 3.14'
fmt.Printf("%05.2f\n", f)      // '03.14'
```

### Строки

```go
s := "Hello"

fmt.Printf("%s\n", s)          // Hello
fmt.Printf("%q\n", s)          // "Hello"
fmt.Printf("%x\n", s)          // 48656c6c6f
fmt.Printf("%X\n", s)          // 48656C6C6F
fmt.Printf("%.5s\n", s)        // Hello
fmt.Printf("%.3s\n", s)        // Hel
```

### Структуры

```go
type Person struct {
    Name string
    Age  int
}

p := Person{Name: "Ivan", Age: 25}

fmt.Printf("%v\n", p)          // {Ivan 25}
fmt.Printf("%+v\n", p)         // {Name:Ivan Age:25}
fmt.Printf("%#v\n", p)         // main.Person{Name:"Ivan", Age:25}
```

## Частые ошибки

### 1. Несоответствие спецификатора и типа

```go
// Некорректный спецификатор для типа
fmt.Printf("%d\n", "Hello")    
// Вывод: %!d(string=Hello)  (программа продолжает работу)

// Правильно
fmt.Printf("%s\n", "Hello")
```

### 2. Пропуск аргументов

```go
// Ошибка! Один спецификатор без аргумента
fmt.Printf("%d %d\n", 42)      // Паника во время выполнения

// Правильно
fmt.Printf("%d %d\n", 42, 100)
```

### 3. Неправильное использование %v

```go
// %v не показывает имена полей
fmt.Printf("%v\n", person)     // {Ivan 25}

// Используйте %+v для имен полей
fmt.Printf("%+v\n", person)    // {Name:Ivan Age:25}

// Используйте %#v для Go-синтаксиса
fmt.Printf("%#v\n", person)    // main.Person{Name:"Ivan", Age:25}
```

## Практические рекомендации

1. **Используйте `%d` для целых чисел** — делает намерение кода явным
2. **Используйте `%q` для строк в выводе** — безопасно для Unicode и специальных символов
3. **Используйте `%v` для отладки** — универсальный спецификатор
4. **Используйте `%+v` для структур** — показывает имена полей
5. **Используйте `%#v` для генерации кода** — вывод в синтаксисе Go
6. **Всегда проверяйте количество спецификаторов и аргументов** — должны совпадать
