# Типы данных в Go

## Примитивные типы

### Числовые типы
- `bool` — логическое значение (true/false)
- `int`, `int8`, `int16`, `int32`, `int64` — целые числа (со знаком)
- `uint`, `uint8`, `uint16`, `uint32`, `uint64` — целые числа (без знака)
- `float32`, `float64` — числа с плавающей точкой
- `complex64`, `complex128` — комплексные числа
- `byte` — псевдоним для `uint8`
- `rune` — псевдоним для `int32` (представление Unicode-символа)

### Строковый тип
- `string` — неизменяемая последовательность байтов (кодировка UTF-8)

## Составные типы

### Массивы и срезы
- `array` — фиксированный размер, например: `[5]int`, `[10]string`
- `slice` — динамический массив, например: `[]int`, `[]string`

### Ассоциативные массивы
- `map` — словарь, например: `map[string]int`, `map[int]string`

### Структуры
- `struct` — пользовательский тип, например:
  ```go
  type Person struct {
      Name string
      Age  int
  }
  ```

### Специальные типы
- `interface` — набор методов, например: `any` (алиас `interface{}` с Go 1.18+), `io.Reader`
- `chan` — канал для goroutines, например: `chan int`, `chan<- string`
- `pointer` — указатель, например: `*int`, `*Person`
- `function` — тип функции, например: `func(int) string`

## Рекомендации по использованию

1. **Используйте `int`** по умолчанию для целых чисел, если размер не критичен
2. **Предпочтение `slice`** над `array` для динамических коллекций
3. **Используйте `map`** только когда нужен быстрый поиск по ключу
4. **Избегайте избыточного использования указателей** — они усложняют код
5. **Используйте `rune`** для работы с отдельными Unicode-символами
6. **Используйте `byte`** для работы с байтовыми данными

## Примеры объявления переменных

```go
// Явное объявление
var age int = 25
var name string = "Ivan"

// Неявное объявление (инференс типов)
var isActive = true
var scores = []int{1, 2, 3}

// Краткая форма (внутри функций)
count := 10
message := "Hello"
```
